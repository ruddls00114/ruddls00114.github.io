# [Vue.js] v-model 이란?

- 데이터 바인딩이란?

  -> Data binding이란 VueJs의 binding expressions을 사용하여 script와 DOM간의 데이터를 주고받는 과정을 의미

- v-model은 Vue에서 양방향 바인딩을 가능하게 한다.

- 단방향 바인딩은 데이터를 화면에 출력

-  양방향 바인딩은 화면에서 입력을 받아 데이터로 다시 전달하는 과정이 추가되어 양쪽 방향 모두 바인딩 되는 것을 의미
